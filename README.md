1. Project Title
Simple Study Planner
What does this project do?
This program helps users create a simple study plan.
Users can add study tasks with subject, topic, duration, and time.
They can also view their tasks and delete tasks when they are completed.ject helps students organize their study schedule by allowing them to add, view, and delete study tasks.

What problem does this project solve?
Many students forget their study tasks or do not plan their study time properly.
This project helps students organize their study schedule by allowing them to add, view, and delete study tasks.
How to run the code?
Step 1: Install Python
Make sure Python is installed on your computer.
Step 2: Save the file
Save the code as:

study_planner.py
Step 3: Open terminal or command prompt
Step 4: Run the program

python study_planner.py
